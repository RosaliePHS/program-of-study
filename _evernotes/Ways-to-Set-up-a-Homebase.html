Sketches:


Why I like Evernote (i.e. what I would need to be able to do with something else if I were to switch) 
  • Phone, iPad and computer apps for writing notes mean you can post from any device easily
  • The web clipper is nice for making good looking posts out of stuff found online
  • The notebooks look nice within the evernote app/the private view looks nice and is user friendly, not just the public view
  • Exporting notebooks into html snippets is easy
  • Tags

Concerns about Evernote:
  • Their public publishing looks terrible 
  • It isn’t free, why pay for a thing when we can build things? 
  • People freaked out when they changed their pricing, and a lot of people seem to abandoning it. Is it doomed? 
  • I like the idea of building my own thing 
  • Inside of Evernote it is not easy to sort by Tags 

Alternatives to Evernote:
  • Joplin seems to be an open source thing designed to be an alternative to Evernote https://opensource.com/article/17/12/joplin-open-source-evernote-alternative
  • I used to love tumblr for this kind of thing, is that even still a thing? 
  • I bet there’s an easy enough way to email posts to a Jekyll site if I want to build my own thing and still be able to work on posts from my phone or iPad 
 
Tools I can use to build a website with it if I want to keep using Evernote to journal:
  • This is a guide to evernote best practices for blogging 
  • Things GitHub other people made to integrate Jekyll and Evernote:
    • https://github.com/chiepomme/chienote--this looks promising, but I don’t completely understand how to set it up. 
    • https://github.com/fooforge/jekyll-evernote_rss--this one warns that it’s buggy and limited in features
  • Postach.io: Made just for this, really simple to set up. BUT the templates they have are really ugly, and you have to pay up to even see what customization is available. 
  • Evernote has an API, so theoretically I should be able to do whatever I want from scratch
  • I bet that I could use the feature of evernote that exports each note as HTML with a resources folder to build up a site using jekyll? 
  • There are Wordpress plug ins for this https://wordpress.org/plugins/evernote-sync/ http://joshpalmeri.com/blog/2017/04/07/evernote-as-a-cms-oh-yes/

TLDR: Basically, if I decide to stick with Evernote, I should be able to use Jekyll or Wordpress to build up a site from it pretty easily. If I don’t stick with Evernote it’s because I can be convinced of a DIY platform that would be comparably easy to post to from multiple devices
